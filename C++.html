This game was created in Urho3D, the basis of the game was an underwater game. The creativity and design were left to me and this is what I came up with.

The game was programmed with C++ and rendered in Urho3D. I used boids to create the fish AI.

This is a clone of space invaders, there are 2D and 3D variations. They were both made with C++.

The 2D version of the game was made using SDL2 whereas the 3D version of the game was made with both SDL2 and OpenGL. The clone doesn't vary from the original game much, the main difference is the 3D version of the game uses a wireframe style to make it look like other retro 3D games like: Battlezone 1980, Akalabeth: World of Doom and others.
